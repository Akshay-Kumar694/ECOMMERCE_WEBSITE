# ECOMMERCE_WEBSITE

 E-commerce Website for Dairy Products 
OBJECTIVE: Develop a full-stack e-commerce platform with Django backend and Bootstrap
 frontend. 
OUTCOME: Responsive web app with user authentication, product catalog, cart/checkout,
 and admin inventory dashboard; solid full-stack experience. 
TECHNIQUES: Django (Python), HTML, CSS, Bootstrap, JavaScript, relational database. 
TOOLS: VS Code.
